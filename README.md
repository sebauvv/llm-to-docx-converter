# LLM MD to Docx Converter

Quick project done for personal use and derusting

https://dqzg7nv7yeylw.cloudfront.net/

## About
Tool for converting LLM chat outputs (like ChatGPT, Claude, Gemini, etc.) to .docx or HTML format

Primarily designed to help maintain formats from long texts or essays generated by LLMs


## Techs used
- Frontend: React + Typescript (Vite)
- Backend: Serverless Lambda Function (Python) 
- Infrastructure: AWS with IaC (Terraform)

## Features
- Markdown editor and live preview
- Download from LLM MD to DOCX or HTML
- Copy HTML for Google Docs / Word
- Light/Dark mode
